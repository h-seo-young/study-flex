## 🗓️ 1일차: 📌 **기본 정렬 - 네비게이션 바 만들기**

### 🔧 요구사항

- 상단 고정형 네비게이션 바
- 가로 방향으로 메뉴 항목 4개
- 가운데 정렬
- 로고와 메뉴를 좌우로 분배
- 배경색: 검정, 텍스트: 흰색

### 📐 명세

```text
+--------------------------------------------+
| Logo           |   Home  About  Contact    |
+--------------------------------------------+
```

- `flex-direction: row`
- `justify-content: space-between`
- `align-items: center`
- 높이 60px, padding 좌우 20px

## 🗓️ 2일차: 🖼️ **사진 갤러리 (Flex-wrap 실습)**

### 🔧 요구사항

- 정사각형 썸네일 9개
- 한 줄에 최대 3개씩, 화면 줄어들면 줄바꿈
- 가운데 정렬
- 썸네일 간의 간격은 16px

### 📐 명세

```text
+-------------------------------+
| [ ] [ ] [ ]                  |
| [ ] [ ] [ ]                  |
| [ ] [ ] [ ]                  |
+-------------------------------+
```

- `flex-wrap: wrap`
- `justify-content: center`
- `gap: 16px`
- 썸네일 크기: 150px × 150px

## 🗓️ 3일차: 📦 **비율 조절 카드 레이아웃**

### 🔧 요구사항

- 가로로 나란히 카드 3개
- 가운데 카드가 양쪽보다 2배 크기
- 카드 간 간격 12px
- 카드 내부는 타이틀과 본문이 세로로 정렬됨

### 📐 명세

```text
+-----------------------------+
| [card1] [   card2   ] [card3] |
+-----------------------------+
```

- `flex-grow`, `flex-shrink`, `flex-basis` 실습
- 내부도 flex 사용 (`flex-direction: column`)
- 각 카드 배경색 다르게

## 🗓️ 4일차: 👤 **프로필 카드 - 순서 바꾸기**

### 🔧 요구사항

- 프로필 사진 + 이름 + 자기소개글
- 데스크탑:

  - 사진 왼쪽, 텍스트 오른쪽

- 모바일:

  - 텍스트 먼저, 사진 아래

- 사진은 원형, 100×100px

### 📐 명세

```text
[Desktop]
+-------------------------------+
| (O)   이름                 |
|       소개글                 |
+-------------------------------+

[Mobile]
+-------------------------------+
| 이름                          |
| 소개글                        |
|     (O)                       |
+-------------------------------+
```

- `order`로 순서 변경
- `align-self`로 사진 정렬 조절
- 반응형 조건: `max-width: 600px`

## 🗓️ 5일차: 🔐 **로그인 폼 중앙 정렬**

### 🔧 요구사항

- 화면 가운데에 로그인 폼
- 폼 크기: 300px
- 내부 구성: 이메일, 비밀번호, 로그인 버튼
- 버튼은 오른쪽 정렬

### 📐 명세

```text
(세로, 중앙정렬)
+----------------------+
| 이메일 입력          |
| 비밀번호 입력        |
|         [로그인]     |
+----------------------+
```

- 부모 요소 높이: 100vh
- `justify-content: center`, `align-items: center`
- 내부는 `flex-direction: column`

## 🗓️ 6일차: 💬 **채팅 앱 UI**

### 🔧 요구사항

- 메시지 풍선 형태
- 좌측: 상대 메시지, 우측: 내 메시지
- 프로필 사진, 이름, 말풍선, 시간
- 말풍선 가로 최대 60%

### 📐 명세

```text
(좌측)
[사진] 이름
┌────────────┐
│  안녕하세요  │ 10:32
└────────────┘

(우측)
      이름 [사진]
      10:33 ┌────────────┐
            │  안녕하세요  │
            └────────────┘
```

- 말풍선 내부도 `flex`
- 전체 메시지 영역은 `flex-direction: column`, 개별 메시지는 `flex-direction: row`
- 말풍선은 `align-self`로 정렬 조절

## 🗓️ 7일차: 🌐 **실전 클론 - Apple 홈페이지 섹션**

### 🔧 요구사항

- Apple Hero 섹션 비슷하게 만들기
- 좌우 양쪽 정렬: 왼쪽은 타이틀, 오른쪽은 이미지
- 가운데 정렬
- 반응형: 모바일에선 위-아래로 배치

### 📐 명세

```text
[Desktop]
+------------------------------------+
|   타이틀 텍스트     [ 아이폰 이미지 ] |
+------------------------------------+

[Mobile]
+--------------------+
| 타이틀 텍스트      |
| 아이폰 이미지      |
+--------------------+
```

- `flex-direction: row` → 모바일에서 `column`
- `justify-content: center`
- 반응형 기준: 768px
